# OpenClaw-MemAdvanced

[中文版](./README_ZH.md) | English

Enhanced personal AI assistant built on OpenClaw with advanced memory capabilities.

## Install

```bash
npm install -g openclaw

openclaw onboard --install-daemon
```

## Quick Start

```bash
openclaw gateway --port 18789 --verbose
```

## About

This is a personal customized fork focused on enhanced memory features.

For the original OpenClaw project, visit: https://github.com/openclaw/openclaw
